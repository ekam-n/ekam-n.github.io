Projects

Volt Legacy

Multi-level challenging platformer where each level focuses on introducing game mechanics, building on them, and then combining mechanics for increased difficulty. Manual implementation of all physics, game-logic, graphics, sound effects, music, and menu GUI.

Controlled Chaos

Here, I had the opportunity to integrate hardware with software. My group and I created an interactive mixed-reality racing game where users control a custom-built RC car which triggers electronic quizzes that the user must complete with a laptop and connected touch sensors.

My role in this project included most of the technical aspects. For example, I was responsible for constructing the RC car and connecting it using Bluetooth and the Arduino microcontroller to a PS5 controller.
