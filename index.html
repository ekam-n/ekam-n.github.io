Projects

Volt Legacy

Project for a course, "Digital Games: Genre, Structure, Programming and Play". This was a fun introductory class where I learned how to implement game physics and basic game design. The class culminated in a final project where I made a multi-level challenging platformer game.

Controlled Chaos

Project for a course, "Introduction to Technological Systems". This was the first class where I had the opportunity to integrate hardware with software. My group and I created an interactive mixed-reality racing game where users control a custom-built RC car which triggers electronic quizzes that the user must complete with a laptop and connected touch sensors.

My role in this project included most of the technical aspects. For example, I was responsible for constructing the RC car and connecting it using Bluetooth and the Arduino microcontroller to a PS5 controller.
